# Privacy Debloater

A lightweight Windows debloating script focused on privacy, telemetry reduction, and removing unwanted default apps.

## Features

- Disables telemetry services.
- Turns off Cortana.
- Removes OneDrive.
- Removes common Windows bloatware.
- Disables ads and suggestions.
- Applies privacy-oriented system settings.

## How to use

1. Open PowerShell as Administrator.
2. Run:

```powershell
.\privacy-debloater.ps1
```

3. Restart if needed.

## Notes

- Review the script before running it.
- Some changes may affect built-in Windows features.
